# Weather Forecast App with Supabase Authentication

This React application provides a 5-day weather forecast for any city using the OpenWeatherMap API. It features user login/logout functionality powered by Supabase, caching and data fetching with React Query, and the ability to toggle temperature units between Celsius and Fahrenheit. The app also saves the last searched city for a better user experience.

---

## Features

- User authentication with Supabase (email/password login)
- Fetch current weather and 5-day forecast from OpenWeatherMap
- Switch between Celsius and Fahrenheit units
- Cache API data using React Query for better performance
- Save last searched city in localStorage for persistence
- Responsive and clean UI with custom CSS
